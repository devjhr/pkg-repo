AndroStudio

PKG Repository

The official package repository for AndroStudio's built-in terminal. Install developer tools directly inside your app — git, python, nano, curl and more. Built for aarch64 Android devices.

Step 1 — Add this repository to your terminal
$ pkg repo add https://devjhr.github.io/pkg-repo

※ Run this once inside AndroStudio's terminal to register the repo.

Step 2 — Install any package
$ pkg install git
$ pkg install python
$ pkg install nano curl

How It Works
1

Open AndroStudio Terminal

Launch AndroStudio on your Android device and open the built-in Termux terminal from the sidebar or terminal tab.

2

Add the Repository

Register this repo so your terminal knows where to fetch packages from.

pkg repo add https://devjhr.github.io/pkg-repo
3

Update Package Index

Fetch the latest package list from all repositories including this one.

pkg update
4

Install Any Package

Install packages by name. Multiple packages can be installed in one command.

pkg install git python nano curl
